ABOUT

Welcome to my version of the Paper Rex merch store. This site is a fan-made project created as part of a web design course. I built it to practice real-world web design skills—everything from layout and styling to navigation and responsive design.

This project is inspired by Paper Rex, one of my favorite esports teams, and I wanted to challenge myself by recreating the look and feel of their official merchandise store. It’s important to mention: This is NOT an official Paper Rex store. It was made purely for educational purposes and is not affiliated with Paper Rex in any way. Nothing on this site is actually for sale.

The goal behind this project was to:

  • Learn how to design a clean, modern e-commerce layout.
  • Practice using HTML, CSS, and possibly other tools like JavaScript or frameworks.
  • Understand how to structure a product catalog and build a user-friendly experience.
  • Develop my skills as a future web designer by working on something I'm passionate about.

If you’re here because you love Paper Rex too—awesome! Just know that this is a practice project created by a student who's learning and experimenting. Thanks for checking it out!
